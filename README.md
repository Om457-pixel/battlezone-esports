# BattleZone Esports Platform 🎮

A full-stack esports tournament platform with an AI-powered analytics engine.

## 🚀 Live Demo
**AI Analytics Dashboard:** https://battlezone-esports-7nps6yjk8yf5xnjlnih85g.streamlit.app

## Projects

### 1. BattleZone App (`esports-tournament-app/`)
Mobile esports tournament platform for Free Fire MAX, PUBG Mobile, COD Mobile.
- Android app (Kotlin + Jetpack Compose)
- Flask + MongoDB backend
- Firebase OTP auth, Razorpay payments, Socket.IO real-time

### 2. AI Analytics Engine (`ai-analytics/`)
AI-powered esports analytics dashboard built on BattleZone data.
- Live leaderboard analytics & player stats
- Win probability prediction (ML)
- AI match commentary & player insights (Groq LLaMA3)
- Interactive charts (Plotly)
